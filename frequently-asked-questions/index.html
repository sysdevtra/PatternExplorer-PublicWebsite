Is it available on my device?

The PatternExplorer is available on Windows. It runs also smoothly together with Parallels on a Mac.

There is a dropdown box on the order page which allows to change the currency to your desired currency.

Is there a trial version available?

Basics Questions

Please make sure that you have carefully read and viewed all relevant documentation. Please send us a request to clarify any questions before purchasing.

Can I add PatternExplorer to my own indicators?

This is not easily possible, however, you can add your own indicators and explorations, filters, buy and sell signals to the PatternExplorer.
There are two files prepared for that: PE – Configuration and PE – Extensions. PE – Configuration is prepared for different scenarios like adding own code before or after PatternExplorer code parts. By adding own code before PatternExplorer code parts you can mix conditions. You can also plot own indicators on certain PatternExplorer indicators.
PE – Extensions is a clean file which allows to quickly add anything to the tools.

The current key has been locked to a different machine

This happens when you run AmiBroker with different user privileges as the PatternExplorer activator. For example AmiBroker with admin rights and PatternExplorer with standard user rights. To solve this just activate again with the same user rights.

AmiBroker doesn't start anymore after installing PatternExplorer

There are usually two possible reasons:

Not activated
The PatternExplorer has not been activated during the 30 day activation period. There is a reminder popping up for 30 days after installation at each AmiBroker start. However, AmiBroker will not start if the reminder is ignored till the end of this period. Remove the PatternExplorer.dll from the plugins folder or activate the software.

Compatibility
New AmiBroker versions, usually beta version, sometimes need an update to our tools as well. We always try to provide an update of our software as soon as possible. In the meantime please inform us and move the PatternExplorer.dll to another folder or install a previous AmiBroker version.

How can I change the theme?

The file PE – Configuration.afl contains the variable PeTheme which can be set to “Light” or “Dark”;

When indicators are inserted the default values are read from the file PE – Configuration. Already inserted indicators can load the new default values by pressing the “Reset All” button in the parameter window.

Can I extend the PatternExplorer?

Yes, own indicators and signals based on these indicators or even combinations of own and built-in indicators are possible.

The customization possibilities are nearly unlimited.

PE - Configuration / PE - Extensions - error message

The PE – Configuration and PE – Extensions files can not be used as an indicator and AFL checks inside the formula will result in errors, because these files are used by all other tools. They can not be used as standalone indicators.

What is the difference between CPR 1 & CPR 2?

The difference between CPR 1 and 2 is the detection method for peaks and throughs. CPR 1 is percentage based while CPR 2 measures the number of bars and distance, similar as the fractal tool.
For rather short term patterns CPR 2 might be better. In most cases it is best to run „Explore“ with both tools to see the different patterns depending on your markets and time frames.
